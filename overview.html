# 概要 ailia DX Insightは、ax株式会社および株式会社アクセルが提供する、企業のDXのためのAIスーパーアプリです。
WindowsやmacOSにインストールするだけで誰でもAIを業務に活用可能です。 ChatGPTの標準機能に加えて、文章検索や画像検索、音声認識、企業内情報検索などailia独自のAIをプラス。 AIを使った業務効率の改善が簡単に行えます。 ![overview_01.png](/img/overview_01.png) ## 機能 aillia DX insightでは下記の機能を使用可能です。各機能で使用されるモデルと実行環境は下記となります。 | 機能 | クラウドで動作 | 端末上で動作 | |:----:|:---:|:----:| | 文章検索（RAG） | ChatGPT、Azure OpenAI、Gemini、Claude(* 準備中)（回答生成） |ailia SDK（インデックス作成、ベクトル検索、Re-Rank機能）、カスタムLLM（回答生成） | | 画像生成 | DALLE | StableDiffusion(* 準備中) | | 要約・アイデア出し | ChatGPT、Azure OpenAI、Gemini、Claude(* 準備中)（回答生成） | カスタムLLM | | 翻訳 | ChatGPT、Azure OpenAI、Gemini、Claude(* 準備中)（回答生成） | ailia SDK、カスタムLLM | | 画像検索 | - | ailia SDK | | 音声入力 | - | ailia SDK | | 音声認識 | - | ailia SDK | | 議事録 | - | ailia SDK | | OCR | Azure AI Vision(* 準備中) | ailia SDK | ## 対応ファイルフォーマット ailia DX Insightがサポートしている形式は下記となります。 ###### 一般的なオフィス系ドキュメントフォーマット `pdf, doc, docx, txt, md, xlsx, csv, tex, css, html` ###### プログラム系ドキュメントフォーマット `c, cpp, json, java, py, rb, php`

`js, swift, .kt, rust, dart, lua`は設定![icon_gear](/img/icon_gear.png)/インデックスから追加することができます。
![overview_02.png](/img/overview_02.png) ###### 画像フォーマット `'png', 'jpg', 'jpeg'` ###### 音声フォーマット `'wav'` ## ファイルの保存先 ailia DX Insightは下記のフォルダにデータベースやモデルファイルを保存します。このフォルダを削除することで、ailia DX Insightを初期化することができます。
`Windows : c:/Users/[UserName]/Documents(*)/ailia DX Insight`
`macOS : ~/Documents(*)/ailia DX Insight`

(*)Windowsでは「マイドキュメント」、Macでは「書類」という呼び方もあります。
